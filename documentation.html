Overview

Peer to Admin - V1 (Chat + Auth)

This type of chat application is tailored for simple websites built with HTML, JavaScript, and CSS, typically used as informational sites. It is ideal for scenarios where the existing product lacks built-in authentication but requires the seamless integration of both authentication and chat functionality.

Instant messaging boosts user engagement, fostering community, satisfaction, and loyalty. It also provides real-time support, allowing users to get quick help. The Chat SDK enables seamless real-time messaging on any app or device.

Prerequisite

To integrate Chat into your app, do the following:

  1. Signup/Login to your account at AddChat.
  2. Create an app as per your use case. (Be Sure to choose the right Mode [Headless | HeaderFul])
  3. Go To created apps's detail page and locate the credentials.

Project setup

To integrate Chat into your app, do the following:

Select the Right Mode and follow the respective integration as below mentioned Steps.

In this mode [HeaderFul], the header will include authentication options (Signup, Login, etc.) along with a chat box for user interaction.

To integrate it into your app, Do the following.

  1. Update the script credentials section with your app's credentials
    1. Locate the main entry file, typically index.html, and insert the following code snippet into the <head> section of the HTML document.
      <script src="https://cdn.socket.io/4.1.2/socket.io.min.js"></script>
    2. <script src="https://cdn.jsdelivr.net/gh/codekeep18feb/addchat-client-cdn-files@v1.0.2/bundle.js"></script>
    3. after script is loaded we can initialize it like below.
      <script>
            document.addEventListener("DOMContentLoaded", async function () {
              const token = localStorage.getItem("tezkit_token");
              if (token) {
                window.chathead.initialize(token);
              } else {
                window.chathead.setUp(
                  "<Your App Name>", 
                  "<Your Api Key>", 
                  JSON.stringify({
                    "header_theme": {
                      "backgroundColor": "rgb(30, 136, 125)"
                    },
                    "chat_opener_theme": {
                      "backgroundColor": "rgb(41, 48, 78)"
                    }
                  }),
                  true,
                  "/index.html"
                );
              }
            });
          </script>
