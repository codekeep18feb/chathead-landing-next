CMS-Ready Chat (Wordpress) [V3]

This chat application is designed specifically for Wordpress using native wp authentication. It is ideal for products that already have built-in authentication probably using some plugin and are looking to seamlessly integrate robust chat functionality.

Prerequisite

To integrate Chat into your app, do the following:
  1. Signup/Login to your account at MagicChat.
  2. Create an app as per your use case. (Be Sure to choose the right version V3)
  3. Go To created app's detail page and locate the credentials.
